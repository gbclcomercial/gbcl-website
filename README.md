
# GBC Auditores - Maqueta Web (Next.js)

Esta es una maqueta profesional y moderna de la página web para GBC Auditores.
Incluye una versión optimizada del logo en `public/logo.png`.

## Requisitos
- Node.js >= 18
- npm o yarn

## Instalación (local)
```bash
npm install
npm run dev
```
Luego abrir `http://localhost:3000`.

## Despliegue rápido (recomendado)
Puedes desplegar gratuitamente en **Vercel** (recomendado para Next.js) o en **Netlify**:
1. Subir el repositorio a GitHub.
2. Conectar el repositorio en Vercel (autodeploy).
3. Listo: la web estará disponible en el dominio que elijas.

Si deseas, puedo preparar el despliegue por ti (subir a Vercel y conectar dominio); dime si quieres que lo haga.
